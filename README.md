### Build and browse e-manual

`preview.sh`実行後にwebブラウザで`http://0.0.0.0:8000/`へアクセスすると
ローカルで編集したマニュアルを閲覧できます。

```sh
$ https://github.com/rt-net/test-emanual.git
$ cd test-emanual
$ ./preview.sh
```
